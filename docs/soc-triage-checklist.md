# Checklist de triage SOC

## Informations de base

- Date/heure de l'alerte
- Source de l'alerte
- Utilisateur concerné
- Hôte concerné
- Type d'événement

## Questions rapides

- Est-ce attendu dans le contexte métier ?
- L'activité est-elle répétée ?
- L'utilisateurice ou l'hôte est-il sensible ?
- Y a-t-il des IOC exploitables ?
- Y a-t-il un impact confirmé ?

## Triage technique

- Vérifier logs endpoint
- Vérifier connexions réseau
- Vérifier authentifications
- Vérifier processus créés
- Vérifier fichiers récents

## Décision

- Faux positif
- Activité suspecte
- Incident confirmé
- Escalade nécessaire
